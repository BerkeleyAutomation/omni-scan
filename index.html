Using a Bimanual Robot with Handoff and Gaussian Splat Merging

Website is currently under construction, interactive videos of our 3D models along with code and paper coming soon!

TL;DR: Omni-Scan uses a bimanual robot to scan objects from multiple viewpoints, then hands off the object to the other arm getting views of previously points. It then merges the scans into a single, visually-accurate 3DGS model.

The website template is adapted from the GARField and Robot See Robot Do project page.
